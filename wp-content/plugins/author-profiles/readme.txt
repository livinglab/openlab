<h2>Showcase the Blog authors with a fun new look to attract the visitors.</h2>
<p> As the name suggests, Fancy WordPress Author plugin widget displays a list of authors on your website using the widget area section.<br>

This plugin has a simple user interface to display author information in the sidebar. All you have to do is drag the author widget into the sidebar and the plugin will do the needful. </p>

<h2>What you can do with this plugin</h2>
<h4>Features</h4>
<small>Some of the feature are available online in pro version.</small>
<ul>
<li>Showcase blog authors in a fun way to attract new visitors</li>
<li>Display a list of authors in the sidebar or on the page</li>
<li>Unique ranking competition among authors</li>
<li>Responsive across all platforms and compatible on all browsers</li>
<li>Upload profile images and social media icons to author profile</li>
<li>Add different effects for a dynamic profile picture (rotate, fade, flip, flash)</li>
<li>Can be displayed using simple shortcodes/ a simple shortcode</li>

</ul>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `authorprofile.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the widget sections to show the profiles author.

== Frequently Asked Questions ==

= How can I display in the sidebar? = 
After activating the plugin, you will see “Author plugin” in WordPress Widget section.

Drag the widget into the required sidebar.

You can customize the setting fields how you want display.

= Do I need to have coding skills to use WordPress Author plugin widget? = 
No. You don’t need any coding skill.
