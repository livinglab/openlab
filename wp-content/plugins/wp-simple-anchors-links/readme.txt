Insert simply many anchors to pages, posts and custom post type.

== Description ==

Simply add anchors links inside your content. Shortcode `[wpanchor id="your_id"]` is added via a button in Wisiwig. After your need to create a link inside your site with #your_id for the href link. Your link look likes `<a href="#your_id">my link</a>`

== Installation ==

1. Upload the plugin wp-simple-anchors-links files to the `/wp-content/plugins/` directory
1. Activate the wp-simple-anchors-links plugin through the 'Plugins' menu in WordPress.
1. Include the shortcode `[wpanchor id="your_id"]` where you want a specific anchor links to be displayed.
